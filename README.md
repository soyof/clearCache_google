# 清理缓存助手 - Chrome 浏览器插件

一个功能强大的 Chrome 浏览器插件，用于快速清理各种类型的浏览器缓存和数据。

## ✨ 功能特性

### 🎯 针对当前网站的操作

- **一键清空当前网站所有缓存** - 仅清理当前访问网站的所有数据
- **清空缓存并重新加载** - 清理当前网站缓存后强制刷新页面
- **当前网站 Cookies** - 删除当前网站的 Cookie 数据
- **LocalStorage** - 清理当前网站的本地存储数据
- **SessionStorage** - 清理当前网站的会话存储数据
- **当前网站 IndexedDB** - 清理当前网站的 IndexedDB 数据库

### 🌐 针对整个浏览器的操作

- **一键清空所有网站缓存** - 清理所有网站的所有类型缓存数据
- **浏览器缓存** - 删除所有网站的缓存文件
- **所有 Cookies** - 删除所有网站的 Cookie 数据
- **所有 IndexedDB** - 清理所有网站的 IndexedDB 数据库
- **历史记录** - 清理浏览历史
- **下载记录** - 删除下载历史记录
- **删除下载文件** - ⚠️ 删除下载的文件（不可恢复）

### ⚙️ 高级选项

- **保留密码** - 清理时保留已保存的密码
- **保留表单数据** - 保留自动填充的表单数据
- **包含受保护数据** - 是否清理受保护的网站数据

## 🎯 使用方法

### 安装插件

1. 下载插件文件到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择插件文件夹

### 界面布局

插件界面分为三个主要模块：

#### 🎯 针对当前网站

- **清空当前网站所有缓存** - 仅清理当前访问网站的数据
- **清空缓存并重新加载** - 清理当前网站并刷新页面
- **分类清理** - 单独清理 Cookies、LocalStorage、SessionStorage、IndexedDB

#### 🌐 针对整个浏览器

- **清空所有网站缓存** - 清理所有网站的缓存数据
- **分类清理** - 浏览器缓存、所有 Cookies、历史记录、下载记录等

#### ⚙️ 高级选项

- **保留密码** - 清理时保留已保存的密码
- **保留表单数据** - 保留自动填充数据
- **包含受保护数据** - 是否清理受保护的网站数据

### 快捷键

- `Ctrl + 1` - 清空当前网站所有缓存
- `Ctrl + 2` - 清空所有网站缓存
- `Ctrl + R` - 清空当前网站缓存并重新加载

### 右键菜单

在任意网页上右键，选择"清理缓存助手"可快速执行常用操作。

## 🔧 技术实现

### 核心技术

- **Manifest V3** - 使用最新的插件规范
- **Chrome Extensions API** - 调用浏览器原生 API
- **Service Worker** - 后台服务处理
- **Content Scripts** - 页面级数据清理

### 清理机制

1. **浏览器级清理** - 使用 `chrome.browsingData` API
2. **页面级清理** - 通过内容脚本直接操作 DOM 存储
3. **Service Worker 清理** - 注销并清理 Service Worker 缓存
4. **Cache API 清理** - 清理 Cache API 存储的数据

### 权限说明

- `storage` - 保存插件设置
- `browsingData` - 清理浏览器数据
- `tabs` - 获取当前标签页信息
- `activeTab` - 操作当前活动标签页
- `downloads` - 管理下载文件
- `history` - 清理历史记录
- `cookies` - 清理 Cookie 数据
- `unlimitedStorage` - 无限制存储权限

## ⚠️ 注意事项

### 安全警告

- **删除下载文件功能** 会永久删除文件，无法恢复
- **清理历史记录** 会删除浏览历史，请谨慎使用
- **清理密码数据** 会删除已保存的密码

### 兼容性

- 支持 Chrome 88+ 版本
- 基于 Manifest V3 规范
- 支持所有现代网站

### 性能优化

- 异步处理，不阻塞浏览器
- 批量操作，提高清理效率
- 智能检测，避免重复清理
